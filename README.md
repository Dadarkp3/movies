## BUILD E EXECUÇÃO
### Running locally

É necessário ter instalado na sua máquina:
- NodeJS, NPM e Angular/Cli

1. Install the dependencies

    ```
    npm install
    
    ```

2. Run the application

    ```
    npm start
    Navigate to http://localhost:4200/
    
    ```
